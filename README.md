# Nait's Reaper Tools
## Installation of Nait's Tools
First make sure you have [ReaPack installed](https://reapack.com/). 

Once it is installed, in Reaper, go to Extensions->ReaPack->Manage Repositories.
* Click "Import/export" button at the bottom, then "Import repositories".
* In the URL box, add the following: https://github.com/naitdev/ReaperTools/raw/main/index.xml
* Click OK, and it should add this repository.
* Double click on Nait ReaperTools in the list of repositories to see tools I've made available.
* You can right click on a tool you're interested in installing, then click "Find in Browser".
* You can also limit the browser by filtering by "Nait" to see what I have created.
* Right click on an item you wish to install, then click Install, then the Apply button.
* Restart Reaper.
